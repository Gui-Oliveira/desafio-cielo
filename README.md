# Desafio Cielo
Desafio feito para realizar a confecção de uma aplicação para ler um arquivo json

## Instalação
A aplicação esta utilizando angular e node.js 12, sendo assim:
- primeiro faça o clone ou o download da pasta do projeto;
- depois de fazer o clone ou o download do projeto, deve entrar na pasta "backend" e ali utilizar "npm i";
- após instalação do backend faça o mesmo procedimento na pasta "frontend".

## Startando a aplicação
Para rodar a aplicação você deve:
- entrar na pasta "backend" e rodar o comando "npm start" (ele deve subir na porta 3001);
- após o start do backend, entre na pasta "frontend" e rode o comando "ng serve" (onde subirá na porta 4200);
- por fim, para visualizar a documentação da aplicação rode o comando "npm run compodoc:serve", assim ele subirá na porta 8080 e ficará disponivel.
